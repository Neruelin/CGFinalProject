Simulating Perfect Astronautical Chore Expeditions project or the SPACE Project for short.

## FOR DEVELOPMENT

### To install dependencies:

    * Install Node.js from https://nodejs.org/en/ - Ensure npm is added to PATH - Run `npm install`

### To start Server:

    * Run `npm run start`

    In another terminal:
    * Run `npm run build`

    * Refresh webpage
